 C++ Mini Games Portfolio

A collection of beginner-friendly console-based games built using C++, developed to strengthen core programming fundamentals such as loops, arrays, functions, and conditional logic. This repository includes classic games like Tic Tac Toe and Snake Game, focusing on logic building and problem-solving skills.

 Features
 Tic Tac Toe (2 Player Game)
 Snake Game (Score-based gameplay)
 Uses C++ fundamentals (arrays, loops, functions)
 Console-based interactive games
 Replay and restart functionality
 Beginner-friendly logic implementation
 
 1. Tic Tac Toe
 Run
g++ tictactoe.cpp -o tictactoe
tictactoe
 Features
3x3 game board
2 players: X and O
Win detection (row, column, diagonal)
Draw detection
Replay option
 Sample Output

  | O |

  | O |

 2. Snake Game
Run
g++ snakegame.cpp -o snakegame
snakegame
Features
Move the snake using W/A/S/D keys
Eat food (*) to increase score
Game over on wall/self collision
Real-time score tracking
Controls
Key	Action
W	Move Up
A	Move Left
S	Move Down
D	Move Right
X	Exit Game
Sample Output
. . . . .
. . O . *
. . . . .
. . . . .
. . . . .

Score: 10
